
# Desarrollo Android - Kotlin: Tarea 1

Proyecto dónde se genera una interfaz gráfica para realizar un registro de usuario solicitando los siguientes datos:

- ✅ Nombre
- ✅ Apellido
- ✅ Género
- ✅ Correo electrónico
- ✅ Contraseña

Adicionalmente, se utilizan los siguientes elementos:

- ✅ TextView
- ✅ ImageView
- ✅ EditText
- ✅ Button
- ✅ RadioButton
